# Mi Primera Web

Este repositorio contiene una aplicación web sencilla. 

![Imagen](https://supercell.com/images/b524ca49e8549e5d3f5485452da7f26c/cropped.webp) 

Practica 25 para practicar con GitHub. 

## ¿Qué hace el proyecto? 

Este proyecto muestra una pagina web sencilla con una página de bienvenida. 

## ¿Por qué el proyecto es útil? 

Es útil para aprender lo básico de cómo usar GitHub. 

## ¿Cómo pueden comenzar los usuarios? 

Los usuarios pueden clonar el repositorio o descargarlo y abrir el archivo `index.html` en su navegador. 

## ¿Dónde pueden recibir ayuda? 

Pueden recibir ayuda revisando la documentación del proyecto o escribiendo en la sección de *Issues* del repositorio. 

## ¿Quién mantiene el proyecto?
